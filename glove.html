PROJECT 03 · EMBEDDED HCI / WEARABLES

Gesture-Controlled Smart Glove

I built a low-cost wearable prototype that uses finger bending and hand orientation for cursor control and air-drawing experiments. The original proposal also considered future VR and robotics control.

REAL BUILD · ITERATIVE PROTOTYPE
01

PROJECT GOAL

I wanted to test a lower-cost alternative to proprietary gesture controllers.

The project started from the cost and limited flexibility of proprietary gesture systems. I wanted an affordable prototype that could be modified for education and experimentation.

I designed the system around finger bending, hand orientation, wireless transmission, and threshold-based gesture mapping. I simplified the design while building it because of the components available.

ORIGINAL PROPOSAL

Five flex sensors + MPU-6050

The original presentation described a five-sensor system intended for VR and robotic control.

→
IMPLEMENTED PROTOTYPE

Two flex sensors + ADXL345

The physical prototype used the available components and focused on testing sensor input, Bluetooth communication, and cursor behavior.

INPUT

Analog flex + motion data

Finger sensors are read from analog inputs while the accelerometer supplies hand orientation.

COMMUNICATION

Wireless serial control

The HC-05 module transmits interpreted motion data to the computer.

EXPERIMENT

Cursor and air drawing

The prototype achieved cursor movement during testing and supported experiments with drawing in space, while exposing calibration and axis-mapping problems.

03

TESTING AND LIMITATIONS

CALIBRATION

Sensor values changed between tests.

Sensor ranges changed with glove fit, hand size, sensor position, and repeated bending. The fixed thresholds required repeated calibration.

DEBUGGING

Bluetooth connection was only one part of the problem.

I had to troubleshoot Bluetooth pairing, Linux rfcomm behavior, Windows serial errors, and coordinate mapping separately from sensor reading.

HCI

The prototype was not comfortable enough for long use.

The weight, wiring, restricted finger movement, signal drift, and fatigue affected usability during testing.

PLANNED IMPROVEMENTS

Use machine-learning classification for gestures.

I plan to improve sensor mounting and calibration, reduce wiring, and test haptic feedback, Wi-Fi, and machine-learning gesture classification.
